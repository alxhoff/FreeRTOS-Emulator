FreeRTOS Simulator Graphical Library

About this library

This basic API aims to provide a simple method for drawing graphical objects onto a screen in a thread-safe and consistent fashion. The library is built on top of the widely used SDL2 graphics, text and sound libraries. The core of the library is the functionality found in TUM Drawing API with extra features such as event handling and sound found in the auxiliary files.
